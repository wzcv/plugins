### 虾米列表型音乐播放器插件Hermit v1.3.1

同名wordpress插件作者亲自定制，编辑面板可自如插入播放器短代码，支持虾米单曲、专辑及歌单的HTML5列表播放。

 > 1.3.1复活版(17-1-27)更新：同步原版核心及虾米API，修正标签参数bug。（[@羽中](https://github.com/jzwalk)）
